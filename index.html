Project list

A list of projects that I have, with a short description. Feel free to browse, clone, submit a PR, or use whatever you like.

It's random, it's different, and it's fun. Coding is life.

  • go-semver is a semantic version library for your go projects. I've used it with clients, and it simplifies having a semantic version consistent with the rest of the world.
  • filebrowser provides abstractions for working with files in PHP. Very basic right now, but has potential. I'm going to be working on this a lot.
  • photogallery is a laravel based web application. I made it so my wife could more easily browse our photo library that is stored on our home server. Has potential to be more.
  • sysmanager was started because I don't like webmin, and don't see any real competition for it yet. Very slowly working on it, with the key to it being a modern design.
  • diceroll parses DND style dice roll strings and rolls the dice. Made in PHP to integrate into other web applications I'm planning on making.
  • go-externalip gets your external ip address (v4 and v6, if available) from behind a nat. Very basic, should probably be made to not rely on external sites.
  • bashext is a basic (very basic) bash plugin system I made to provide some more useful commands. Don't really use it much anymore, but it exists and maybe it's useful.
